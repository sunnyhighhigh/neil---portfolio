University Project: Safety-Critical Medicine Infusion System

Designed a safety-critical medicine infusion system using a heterogeneous architecture that combined an FPGA soft-core, an NXP LPC1763 (ARM Cortex-M3), and a legacy 8051. By using different hardware platforms, the design reduces common mode failures where a single processor or compiler issue could affect the whole system.

The NXP LPC1763 (Cortex-M3) acted as the main application processor handling infusion-process complexity, the FPGA soft-core (NIOS II) handled deterministic real-time hardware interface control, and the 8051 microcontroller served as an independent safety monitor/watchdog.

Safety-Critical Design | ARM Cortex-M3 | NIOS II FPGA Soft-Core | 8051 | Embedded Systems
